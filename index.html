How has energy by source changed over time?

This section explores how Australia’s energy consumption has evolved by source over time. It highlights the shift away from fossil fuels toward renewable energy, showing clear upward trends in solar and wind generation since 2015. The interactive line chart allows users to hover for precise values and use the range selector to focus on specific decades or policy-driven periods of change.

How to read: Lines = sources; higher line = more generation (TWh). Use the year slider; hover for year/source values.

Electricity Consumption by Australian State

This choropleth map visualizes how electricity use varies across Australia’s states and territories. Darker shades indicate higher consumption, which is typically concentrated in more industrialized and densely populated regions like New South Wales and Victoria. The map helps contextualize national energy trends geographically, offering a snapshot of regional demand patterns.

How to read this chart
  • Colour encodes total consumption (MWh): darker = higher.
  • Hover a state to see its name and value.
  • Tip: Adjust the page’s year controls (if available) to compare periods.

Energy Mix by Source (Selected State)

This donut shows the share of electricity generated by each source for the selected region (use the State menu to switch between states or AUS). It provides a quick snapshot of the latest year’s mix across coal, gas, oil, renewables and hydro.

How to read this chart
  • Slice size = the source’s percentage of total generation.
  • Hover a slice to see the source name and exact share.
  • Change “State” to compare mixes across regions.
  • Tip: Use with the time-series chart to see how this mix has evolved.

Production vs Consumption of Electricity by States

This scatter plot compares each state’s electricity production with its total consumption. The diagonal reference line (x = y) marks the point where production equals consumption points above the line represent states that consume more than they produce, while points below indicate net producers.

How to read this chart
  • X-axis = production (TWh), Y-axis = consumption (TWh).
  • Distance from the line shows surplus or deficit size.
  • Hover to see the state and exact values.

Global Comparision of Electricity Consumption

This view puts Australia in context by showing per-capita electricity use (kWh/person) worldwide. The map shades each country by its per-person use, and the bar chart ranks the top 30 users. Resource- or industry-intensive economies (e.g., Qatar, Singapore, Iceland) tend to sit near the top, while many large-population countries rank lower on a per-person basis despite high total demand.

How to read: Map — colour shows kWh/person; darker = higher; hover for country and value. Bar chart — x-axis is kWh/person; bars are ordered from highest to lowest; hover for exact figures. Note — values are per person (not total consumption) and reflect the latest year in the dataset.
